P4 Workshop
========================================

P4 Workshop, a collection of useful tools for P4 developments

**Contents**
- [Features](#features)
- [Build \& Install](#build--install)


Features
----------------------------------------

- *P4WS Build*: Cross-target cmake build system for P4 programs, see [Build P4 Program](./docs/build-p4-program.md).
- *P4WS P4 Includes*: Includes for P4 programs.
- *P4WS Python Package*: Python package provides tools for P4 development:
  - Load mininet topology from P4 program.
  - Patch P4 SDEs.
  - [Transfer P4 programs](./docs/transfer-p4-program.md).
